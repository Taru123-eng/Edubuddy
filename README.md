# Edubuddy

**Edubuddy** is an AI-powered learning companion designed to make education more accessible, interactive, and personalized for students in low-resource settings. It provides summarized lessons, automated MCQ generation, and progress tracking‚Äîall within an intuitive and engaging platform.

##  Features

- **Lesson Summarizer**  
  Upload or input lesson content, and EduBuddy‚Äôs AI instantly generates concise, easy-to-understand summaries.

- **AI-Generated MCQs**  
  Automatically create multiple-choice questions from any text to test comprehension and reinforce learning.

- **Simple Interface**  
  Designed for ease of use, especially for younger learners or those with limited tech experience.

-**Optimized for Light Mode**
  EduBuddy is designed to function and appear best in light mode for maximum readability and a clean learning experience.

##  Project Objective

EduBuddy was developed to bridge the educational gap for students lacking access to personalized learning tools. By integrating AI summarization and automated assessment, it aims to support teachers and NGOs in delivering scalable, quality education.
